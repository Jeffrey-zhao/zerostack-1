零度框架是一套基于微服务和领域模型驱动设计的企业级快速开发框架，基于微软.NET 5 最新技术栈构建，容器化微服务最佳实践，几十种开源框架的优雅组合，支持模板自动化生成机制，完善认证与授权管理机制，通用日志组件，异常处理组件，缓存组件，依赖注入组件，模型映射组件和验证组件等，该框的目标是提高用户体验的同时，致力于成倍提高开发人员的工作效率。
